Blank Page
==========

Entry in PyWeek #10  <http://www.pyweek.org/10/>
Team: Glup
Members: gravy_lips, Urser, pufuwozu


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  Pyglet:     http://www.pyglet.org/



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Otherwise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Use the WASD keys to move around the world.

Aim with the mouse and press the left button to shoot goo.
